This guide defines the following instances of the Questionnaire and QuestionnaireResponse profiles. The Questionnaire instance defines the HAI form and the QuestionnnaireResource instance is an example of a completed QuestionnaireResponse to the Questionnaire.

Instance Name Description
Questionnaire/hai-questionnaire-los-event This Questionnaire instance defines the questions and possible answers in the Late Onset Sepsis/Meningitis Event (LOS) Report.
Questionnaire/hai-questionnaireResponse-los-event This QuestionnaireResponse instance is an example of a completed Late Onset Sepsis/Meningitis Event (LOS) Report and is in response to the Questionnaire instance above.
